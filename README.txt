Steps to run the app it should be run on localhost .

1. login {username:Admin, password:12345}
2. Test API's
3.API are as follows
//LOGIN 
http://localhost:5000/api/auth/login

//CREATE COURSE
http://localhost:5000/api/courses/add/course

//UPDATE COURSE
http://localhost:5000/api/courses/{pass course id here}

//DELETE COURSE
http://localhost:5000/api/courses/delete/course/{pass course id here }

//GET ONE COURSE
http://localhost:5000/api/courses/find/{pass course id here }

//GET ALL COURSES
http://localhost:5000/api/courses/getall/
